# Dobboxให้และรับข้อคิดเห็น
Dropbox ทำให้การส่งไฟล์ขนาดใหญ่และขนาดเล็ก รวมถึงการรวบรวมข้อคิดเห็นจากทุกคนไว้ในที่เดียวเป็นไปได้ง่ายดาย ผู้ชมสามารถเข้าถึงลิงก์และตัวอย่างไฟล์กว่า 150 ประเภท และยังสามารถเพิ่มความคิดเห็นและอ่านความคิดเห็นของคุณได้แม้จะไม่มีบัญชี Dropbox ก็ตาม
ได้เวลาของคุณกลับคืนมา
แอป Dropbox สำหรับอุปกรณ์เคลื่อนที่ทำให้งานปกติทั่วไปที่กินเวลาในแต่ละวันของคุณง่ายดายขึ้น สแกนใบเสร็จรับเงิน กระดาษโน้ต และเอกสารอื่นๆ ได้ในไม่กี่วินาทีโดยใช้กล้องโทรศัพท์ของคุณ ประหยัดเวลาและหายห่วงด้วยการสำรองรูปภาพและวิดีโออัตโนมัติ
